Git

Git is a Version Control System. You can push your gits to servers online like github. Git is just a fancy save button.

GitHub

Git hub makes it easy to store your repositories.

Git Commits

It is best practice to follow the conventions when making your git commits. Be sure to follow these rules:

  • Use a subject line, keep it at 50 characters
  • Use the imperative voice (as a command)
  • Use the body to explain what & why, not how
